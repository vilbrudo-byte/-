# Сайт-презентация подарка

Минималистичный адаптивный сайт для презентации подарка - поездки в загородный отель "Чайка".

## Структура проекта

- `index.html` - главная страница с кнопкой "Открыть подарок"
- `gift.html` - страница с полной программой мероприятия
- `style.css` - стили для обеих страниц
- `фото/` - папка для локальных изображений (если потребуется)

## Использование изображений

Сейчас сайт использует изображения напрямую с сайта отеля:
- https://chayka-hotel.ru/wp-content/uploads/2022/04/front-side-hotel.jpg
- https://chayka-hotel.ru/wp-content/uploads/2022/04/standar-room.jpg
- https://chayka-hotel.ru/wp-content/uploads/2022/06/spa-1.jpg
- https://chayka-hotel.ru/wp-content/uploads/2022/06/restourant.jpg

### Если нужно использовать локальные изображения:

1. Скачайте изображения с сайта отеля
2. Поместите их в папку `фото/`
3. Замените URL в файле `gift.html` на локальные пути:
   ```html
   <img src="фото/hotel.jpg" alt="Фасад отеля">
   <img src="фото/room.jpg" alt="Номер Стандарт">
   <img src="фото/spa.jpg" alt="СПА-зона">
   <img src="фото/restaurant.jpg" alt="Ресторан">
   ```

## Запуск

Просто откройте файл `index.html` в браузере. Сайт полностью адаптирован для мобильных устройств.

**Даты:** 17-19 декабря 2024

**Включено:**
- Проживание в номере категории "Стандарт"
- Трёхразовое питание
