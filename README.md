# Сайт-презентация подарка

Минималистичный адаптивный сайт для презентации подарка - поездки в загородный отель "Чайка".

## Структура проекта

- `index.html` - главная страница с кнопкой "Открыть подарок"
- `gift.html` - страница с полной программой мероприятия
- `style.css` - стили для обеих страниц
- `фото/` - папка с зимними изображениями отеля

## Запуск

Просто откройте файл `index.html` в браузере. Сайт полностью адаптирован для мобильных устройств.

## Программа мероприятия

**Даты:** 17-19 декабря 2025

**Включено:**
- Проживание в номере категории "Стандарт"
- Трёхразовое питание
- СПА-услуги (бассейн, сауна, массаж)
- Зимние развлечения: каток, лыжи, тюбинг, хоккей
- Боулинг и прокат зимнего инвентаря
